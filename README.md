## 🏗️ Architecture

<div style="margin: 30px 0; padding: 25px; border-radius: 15px; background: linear-gradient(135deg, #ffffff, #f5f5f5); box-shadow: 0 8px 20px rgba(0,0,0,0.05);">
  <p>The application follows a clean architecture approach with:</p>
  
  <ul style="list-style-type: none; padding: 0;">
    <li style="margin: 10px 0; padding-left: 30px; position: relative;">
      <span style="position: absolute; left: 0; color: #0D8ABC;">▶</span>
      <strong>Screens:</strong> UI components organized by feature
    </li>
    <li style="margin: 10px 0; padding-left: 30px; position: relative;">
      <span style="position: absolute; left: 0; color: #6A11CB;">▶</span>
      <strong>Models:</strong> Data structures for AI bots, knowledge items, prompts, etc.
    </li>
    <li style="margin: 10px 0; padding-left: 30px; position: relative;">
      <span style="position: absolute; left: 0; color: #00B359;">▶</span>
      <strong>Providers:</strong> State management using Provider pattern
    </li>
    <li style="margin: 10px 0; padding-left: 30px; position: relative;">
      <span style="position: absolute; left: 0; color: #FF9900;">▶</span>
      <strong>Services:</strong> Business logic and API integration
    </li>
    <li style="margin: 10px 0; padding-left: 30px; position: relative;">
      <span style="position: absolute; left: 0; color: #DC143C;">▶</span>
      <strong>Widgets:</strong> Reusable UI components
    </li>
    <li style="margin: 10px 0; padding-left: 30px; position: relative;">
      <span style="position: absolute; left: 0; color: #4B0082;">▶</span>
      <strong>Utils:</strong> Helper functions and utilities
    </li>
  </ul>
</div>
